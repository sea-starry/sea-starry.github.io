python实现简单计算平均分

Posted on 2019-09-10

实现代码：

from tkinter import *
from tkinter import messagebox
class AverageOfgrade:
    def __init__(self):
        win = Tk()
        win.title("学生成绩统计")

        grade1 = Label(win,text="C++程序设计")
        grade1.grid(row=0,column=0,pady=5,padx=5,sticky="e")
        self.number1 = IntVar()
        entNumber1 = Entry(win,width=15,textvariable=self.number1)
        entNumber1.grid(row=0,column=1,pady=5,padx=5,sticky="w")

        grade2 = Label(win,text="C++程序设计")
        grade2.grid(row=1,column=0,pady=5,padx=5,sticky="e")
        self.number2 = IntVar()
        entNumber2 = Entry(win,width=15,textvariable=self.number2)
        entNumber2.grid(row=1,column=1,pady=5,padx=5,sticky="w")

        bt_Average = Button(win,text="计算平均成绩",command=self.Average)
        bt_Average.grid(row=4,column=0,columnspan=2,pady=5)

        result = Label(win,text="平均成绩：")
        result.grid(row=5,column=0,padx=5,pady=5,sticky="e")
        self.result_number = DoubleVar()
        getResult = Entry(win,width=15,state="readonly",textvariable=self.result_number)
        getResult.grid(row=5,column=1,padx=5,pady=5,sticky="w")
        win.mainloop()

    def Average(self):
        g1 = float(self.number1.get())
        g2 = float(self.number2.get())
        sum =(g1 + g2)/2
        self.result_number.set(round(sum,2))

Python实现简单数据库查找

Posted on 2019-09-04

思路

  1. 将所需要录入的学号、姓名保存到text文件里。
  2. 编程打开自己保存的text文件，并用数组储存保存的信息。
  3. 选择自己是输入学号还是姓名。
  4. 如果输入的是学号，从第一行开始查找，找出与输入学号相同的一行，输出所需要的信息，否则查找下一行；输入姓名和学号进行相同的操作。
